# fmkorea 스크래퍼 TDD 기반 구현 계획

## 작업 목표
- TDD 기반으로 fmkorea 스크래퍼 구현
- 게시글 목록 및 상세 페이지 스크래핑 기능 구현
- 스크래핑 데이터를 JSON 형식으로 저장

## 작업 단계
1. [ ] 테스트 환경 설정
   - [ ] 필요한 라이브러리 설치
   - [ ] 테스트 파일 생성

2. [ ] 스크래퍼 기본 구조 테스트 작성
   - [ ] 스크래퍼 클래스 인터페이스 테스트
   - [ ] 기본 메서드 테스트

3. [ ] 게시글 목록 스크래핑 테스트 작성
   - [ ] 게시글 목록 파싱 테스트
   - [ ] 게시글 메타데이터 추출 테스트

4. [ ] 게시글 상세 페이지 스크래핑 테스트 작성
   - [ ] 게시글 내용 파싱 테스트
   - [ ] 댓글 파싱 테스트

5. [ ] 스크래퍼 구현
   - [ ] 스크래퍼 클래스 구현
   - [ ] 게시글 목록 스크래핑 구현
   - [ ] 게시글 상세 페이지 스크래핑 구현

6. [ ] 리팩토링
   - [ ] 코드 최적화
   - [ ] 예외 처리 추가

7. [ ] 실제 데이터 스크래핑 및 검증
   - [ ] 첫 페이지 게시글 스크래핑
   - [ ] JSON 파일로 저장
   - [ ] 결과 검증

## 사용할 기술 및 라이브러리
- Python 3.8+
- BeautifulSoup4 / Playwright
- pytest
- MongoDB
- jsonschema
